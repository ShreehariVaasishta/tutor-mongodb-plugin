tutormongo plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/ShreehariVaasishta/tutor-mongodb-plugin.git

Usage
-----

::

    tutor plugins enable tutormongo
